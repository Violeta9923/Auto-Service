Pentru testare:
    Utilizatorul poate adauga un angajat sau o masina, urmand
meniul de comenzi afisat la consola. Pentru a putea adauga o masina
trebuie mai intai adaugat un angajat.

    Masinile au fost identificate printr-un numar:
- 0 : masini standard
- 1 : autobuze
- 2 : camioane

    Timpul estimat de un angajat pentru repararea unei masini
este impartit astfel:
- 0.2 pentru masinile standard
- 0.4 pentru autobuze
- 0.6 pentru camioane
